Agentic Compute on GPU

This evolving Python framework orchestrates AI agents for Triton, CUDA, CuPy, and cuDF to achieve kernel autotuning and optimization. It follows the Sense → Think → Act → Learn loop, integrating profilers (Nsight) and leveraging advanced GPU primitives to create a single, knowledge-driven GPU research layer.
